# AI-Text-Analyzer
AI-Powered Text Analyzer

>An interactive NLP-based web application that analyzes text and provides meaningful insights in real time.

 Key Features:

Sentiment Analysis (Positive / Neutral / Negative)
Keyword Extraction & Frequency Analysis
Word Cloud Visualization
Downloadable Analysis Report
Clean and interactive UI

 Tech Stack:
Python, Streamlit, TextBlob, Matplotlib

 About the Project

This project demonstrates how Natural Language Processing (NLP) can be used to extract insights from text data. It processes user input, performs sentiment analysis, identifies key terms, and visualizes the results through charts and word clouds in an intuitive web interface.
